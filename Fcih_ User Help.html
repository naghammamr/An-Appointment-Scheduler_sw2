User Help

Scheduler

As a Student:

i. Login:

          1- Open the site

          2- Enter username and password

          3- Click login

 

ii. Choose Course

1-    Select course you want.

2-    Click scheduler name.

 

iii. Select Slot

 

1-    Book slot you want from available slots for appointment.

2-    You can cancel after booking.

 

As a Teacher:

i. Login:

          1- Open the site

          2- Enter username and password

          3-    Click login


ii. Choose Course

           1-    Select course you want to add schedule to.

           2-    Click scheduler name.

 

iii. Add Slots

            1-    Click Add slots.

            2-    Choose single slot or repeated slots.

            3-    Fill the form of slots then click save changes.

            4-    Teacher can edit the slots options by clicking in ().

 

iv. Delete Slots

             1-    Click delete slots if you want to delete slots.


iiv. Export

             1- Click export.

             2- Fill the form.

             3- Click export file , this will lead to generate file filled with form choices.
